# Checkout

Simple steps for an easy departure.

## Quick version
- Checkout time: see your Airbnb booking for the exact time.
- Tidy up: start the dishwasher, and put used towels in the tub or laundry basket.
- Trash to outdoor bins by the driveway; recycling in the blue bin.
- Set thermostat to a reasonable temp (around 68°F heat / 74°F cool).
- Close and lock all doors and windows; return any moved items to original spots.
- Message us on Airbnb when you’re on your way out.

## Details
- **Kitchen:** Load and start the dishwasher with any remaining dishes. Wipe counters and toss perishable food.
- **Laundry:** Place used towels in the tub or laundry basket; no need to start laundry unless you’d like to help.
- **Trash & recycling:** Bag all trash and place in the outdoor bin. Clean recyclables can go in the blue bin; no plastic bags.
- **Climate:** Set thermostat to 68°F if heating season or 74°F if cooling season. Turn off portable heaters or fans.
- **Furniture:** Return moved furniture or cushions to their original positions to avoid damage.
- **Keys/access:** Ensure the smart lock is fully engaged and any spare keys/fobs are left on the kitchen counter.

## FAQs
- **Can I check out late?** Please ask in Airbnb; we’ll confirm if cleaning schedule allows.
- **Do I need to strip beds?** Not required, but a quick strip is appreciated if you have time.
- **What if I forgot something?** Message us with a description; we’ll look during turnover and coordinate shipping at cost.
